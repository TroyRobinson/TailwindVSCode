LLM Tailwind Class Editor Test

How to use:

  1. Open this file in VS Code
  2. Click the "Tailwind: Open Preview" button in the top toolbar
  3. Double-click on any element in the preview
  4. In the popup editor, type your classes followed by " -- " and a prompt
  5. Example: p-4 bg-blue-500 -- make the padding larger
  6. Press Enter or click "Send to LLM" to edit the classes
  7. Watch the console (F12) for debugging logs
